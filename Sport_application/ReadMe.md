Maquette d’une application mobile sociale dédiée aux sportifs, réalisée dans le cadre du concours Campus Création. 
L’application permet de proposer ou de rejoindre des activités sportives entre amis, de visualiser les événements à 
proximité sur une carte, d’échanger avec d’autres utilisateurs et de consulter les profils. Le prototype a été développé avec Flutter (Dart) 
afin de cibler à la fois Android, iOS et le web.


Prototype of a mobile social application for athletes, developed as part of the Campus Création competition. The app allows users to create or join 
sports activities with friends, view nearby events on a map, chat with other users and browse profiles. The prototype was built with Flutter (Dart) 
to target Android, iOS and the web from a single codebase.
